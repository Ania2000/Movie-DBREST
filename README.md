# FastAPI – Movies API
Prosta aplikacja REST API napisana w FastAPI

## Opis projektu 
Aplikacja udostępnia:
- proste endpointy matematyczne,
- endpoint do reverse geocodingu (OpenStreetMap / Nominatim),
- REST API do zarządzania listą filmów zapisaną w bazie SQLite.

### Wymagania 
- Python 3.10+
- pip
- PyCharm (zalecany)# Movie-DBREST
